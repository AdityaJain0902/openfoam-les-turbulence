# OpenFOAM Turbulence Modelling

M.Tech Thesis Project at IIT Gandhinagar

## Overview

This repository contains my work on computational fluid dynamics (CFD) using OpenFOAM, focusing on turbulence modelling, numerical validation, and Large Eddy Simulation (LES).

The project aims to develop a strong understanding of turbulence physics, validate numerical methods against benchmark cases, and build towards high-fidelity LES simulations of turbulent flows.

## Cases Studied

### 1. Lid-Driven Cavity Flow
- Solver: icoFoam
- Reynolds Number: Re = 100
- Structured hexahedral mesh
- Validation against Ghia et al. benchmark data
- Velocity profile comparison along cavity centrelines

### 2. Flow Over a Cylinder
- Solver: icoFoam
- Reynolds Numbers: Re = 2, 10, 40, 200, 1000
- Pressure and velocity field analysis
- Wake development and flow separation study
- Drag coefficient verification against published reference data

### 3. Turbulence Modelling Fundamentals
- Boundary layer theory
- Wall treatment and y+ concepts
- Viscous sublayer, buffer layer, and log-law region
- Introduction to RANS and LES methodologies
